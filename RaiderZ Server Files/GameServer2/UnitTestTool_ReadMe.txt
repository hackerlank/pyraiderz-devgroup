1. 현제 지원되는 기능
   - 실행할 TEST와 SUITE의 이름을 단축키로 입력 및 제거 할수 있음
   - 처음 실행 했을때로 실행할 TEST와 SUITE의 이름을 되돌릴수 있음

2. 클립보드에 실행할 TEST나 SUITE의 이름을 복사한뒤 사용할 수 있는 단축키
   - CTRL + Insert : 비어있는 실행할 TEST 항목에 입력한다.
   - CTRL + 1~4 : 지정된 숫자의 TEST에 입력한다.
   - CTRL + Delete : 실행할 TEST를 모두 제거 한다.

   - ALT + Insert : 비어있는 실행할 SUITE 항목에 입력한다.
   - ALT + 1~4 : 지정된 숫자의 SUITE에 입력한다.
   - ALT + Delete : 실행할 SUITE를 모두 제거 한다.

3. 버튼
   - 닫기 : 프로그램 종료
   - 초기화 : 프로그램을 처음 실행 했을때로 TEST와 SUITE를 되돌린다.
